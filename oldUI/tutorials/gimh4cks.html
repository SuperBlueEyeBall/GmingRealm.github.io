Gimkit h4cks Tutorial

This version of Gimkit Cheat was inspired by Gimkit Utility by UndercoverGoose, which was sadly taken down. The main things that were carried over from the other script are using Typescript, Rollup and Tampermonkey, which makes it easier to develop and use.

How to install

Install/Update it by clicking on this link while having a userscript manager installed on your browser.

How to get a userscript manager

Option 1: Tampermoney

Simply go to the Official Tampermonkey Download Page, select your browser on the second top row, and follow the instructions on how to install it for your browser down below.

Option 2: What If Tampermonkey is disabled in the extension store?

There is an easy fix for that! Tampermonkey is by far the most popular userscript manager, but you can use others one if you want or if Tampermonkey is blocked/disabled. For example, I also use Violentmonkey as you can’t use Tampermonkey in a mobile browser such as kiwi browser.

Disclaimers (important)

Disclaimer for Chromebooks users

The GimKit h4cks work fine on high quality computers such as windows operating systems on computers in a computer science class. If you use these on a chromebook, it will likey be very laggy and you may not even be able to enter the game. Please continue reading on the next sub-page.

Read this for all devices

It is sometimes hard to join a game and have it register. If you want to have a higher change of being able to join, then type in your name as quick as possible and press join. Make sure you copy the join URL or just refresh as the h4cks will have already loaded in.

Features

Gimkit Cheat provides a nice, unified way to use different cheats. To open/close the hud, press “\” (backslash) on your keyboard while on Gimkit. It will show various menus, which should be fairly self explanatory. If you find any issues, please open a new issue on this repository.

General Cheats

  • Auto Answer: Automatically answers questions for you.

  • Cosmetic Picker: Allows you to use any cosmetic you want, even unused ones. These are only visible to you.

  • Freecam: Allows you to move your camera wherever you want, or even spectate other players. Once in freecam, use the arrow keys to move the camera.

  • Player Highlighter: Marks on your screen where teammates or enemies are in relation to you.

  • Instant Use: Tired of waiting for painfully slow bars to fill up just to purchase something minor? This allows you to use the nearest thing instantly by hitting enter.

Gamemode Specific Cheats

  • Classic: Automatically purchases upgrades for you. Best used with Auto Answer.

  • Super Rich Mode: Identical to Classic, with adjusted values.

  • Trust No One: Tells you who the imposters are. Doesn’t work if you join mid-game.

  • Capture The Flag: Purchase upgrades from anywhere

  • Tag: Purchase upgrades from anywhere

  • Snowbrawl: Purchase shield cans/medpacks from anywhere

  • One Way Out: Purchase shield cans/medpacks from anywhere

  • Farmchain: Adds options to automatically harvest/plant crops from anywhere
